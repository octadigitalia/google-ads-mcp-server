## Objective

Transform the MCP from a proof-of-concept into a final user-ready product, focusing on token economy, robust resource resolution, and simplified setup.

---

## Business Value

- Drastically reduces operating costs (API tokens) by delegating technical resolution to the server.
- Improves user experience by removing technical barriers (IDs, complex credential setup).
- Increases system reliability by preventing ID hallucinations.

---

## Scope

### In Scope
- Dynamic resource resolution (Name -> ID) for Campaigns, AdGroups, and UserLists.
- AI Skills Companion for orchestrated workflows.
- Simplified onboarding / Setup Wizard.
- Error translation layer for human-friendly messages.

### Out of Scope
- Automatic Google Cloud project creation (must be manual).
- Billing payment automation.
