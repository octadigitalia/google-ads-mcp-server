# Story 9.2: Aggregated Account Snapshot (The AI Dashboard)

**Status:** Done

## Story
**As an** AI Agent,
**I want** a single tool that returns a high-level performance and status snapshot of the entire account,
**so that** I don't have to call multiple tools (`get_performance`, `list_customers`, `get_capabilities`) to understand the initial state of the project.

## Acceptance Criteria
1. [x] Create `get_account_snapshot(customer_id, date_range)` tool in `server.py`.
2. [x] **Aggregated Metrics**: Include Total Clicks, Conversions, Cost, and ROAS for the account in the period.
3. [x] **Top Campaigns**: List the top 5 campaigns by spend/conversions.
4. [x] **Status Health**: Return counts of ENABLED vs PAUSED campaigns/adgroups.
5. [x] **Optimization Score**: Include the account-wide optimization score (if available via `Customer` resource).
6. [x] **Context Density**: Use a dense JSON format that removes all null/default values to save tokens.
