## Story
**As an** AI Agent,
**I want** to query the `GoogleAdsFieldService` to validate my GAQL fields before executing a search,
**so that** I can avoid API errors and learn the correct schema dynamically.

## Acceptance Criteria
1. [ ] Implement a helper that queries `google_ads_field` resource.
2. [ ] Validate if a list of fields is compatible with a given resource (e.g., are these metrics allowed for 'campaign'?).
3. [ ] Integrate this validation into the `search_ads` error handling flow.
4. [ ] Return a list of valid fields when an invalid one is detected.

## Tasks
- [ ] Task 1: @architect: Map the `google_ads_field` metadata structure.
- [ ] Task 2: @dev: Implement the `validate_gaql_fields` helper.
- [ ] Task 3: @dev: Update `search_ads` to use the helper on failure.
