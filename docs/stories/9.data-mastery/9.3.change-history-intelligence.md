# Story 9.3: Change History Intelligence (Audit Trail)

**Status:** Done

## Story
**As an** AI Agent,
**I want** to see recent changes made to the account (by humans or other agents),
**so that** I can correlate performance shifts with specific actions and avoid undoing recent manual optimizations.

## Acceptance Criteria
1. [x] Create `get_change_history(customer_id, last_n_days)` tool in `server.py`.
2. [x] **Resource Visibility**: Detect changes in Campaigns, AdGroups, and Keywords.
3. [x] **Action Attribution**: Identify if the change was an ADD, UPDATE, or REMOVE.
4. [x] **Contextual Detail**: Show which fields were changed (e.g., Budget changed from 100 to 120).
5. [x] **Dense Output**: Filter out system-generated or noise changes to focus on user-intent actions.
