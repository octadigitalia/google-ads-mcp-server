## Story
**As a** New User,
**I want** an interactive setup script that guides me through the Google Ads API complexity,
**so that** I can start using the MCP by providing my own credentials through a guided process.

## Acceptance Criteria
1. [x] Create an interactive CLI tool (`python mcp-setup.py`).
2. [x] **Automated OAuth2 Flow**: Implement a local callback server to capture authorization codes automatically (eliminating manual copy-paste).
3. [x] **In-CLI Documentation**: Provide links and step-by-step instructions inside the terminal for creating a Google Cloud Project and obtaining a Developer Token.
4. [x] **Strict Credential Validation**: Test user-provided credentials and suggest specific fixes for common errors (Invalid Secret, Unauthorized Token, etc.).
5. [x] **Environment Sync**: Automatically populate `.env` and `google-ads.yaml`.
