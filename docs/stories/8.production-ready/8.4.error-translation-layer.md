## Story
**As a** Google Ads AI Agent,
**I want** technical gRPC and Google Ads API errors to be translated into semantic, human-readable instructions,
**so that** I can understand exactly what went wrong and attempt to fix the error (Self-Healing) without user intervention.

## Acceptance Criteria
1. [x] Create `translate_google_ads_error(exception)` in `utils.py`.
2. [x] **Invalid Query (400)**: Map GAQL syntax errors to suggestions (partially covered by Linter, but should handle other 400s).
3. [x] **Permission Denied (403)**: Distinguish between "Unauthorized Customer", "Expired Token", and "Permission Denied on Resource".
4. [x] **Policy Violation**: Catch `PolicyViolationError` and provide a clear "ACTION_REQUIRED" message explaining which field violated which policy.
5. [x] **Resource Not Found**: Standardize 404-like errors for resolved names that don't exist in the API context.
6. [x] Apply the translator to all mutation tools (`set_campaign_status`, `create_search_campaign`, etc.).
