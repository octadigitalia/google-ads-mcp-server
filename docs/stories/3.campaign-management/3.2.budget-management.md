# Story 3.2: Gerenciamento de Orçamentos Diários

**Epic:** 3 (Campaign Lifecycle)
**Status:** Done
**Role:** Developer
**Action:** Implementar a tool `set_campaign_budget` para ajuste dinâmico de investimento
**Benefit:** Permitir que o sistema escale orçamentos de campanhas com boa performance ou limite gastos em campanhas ruins

---

## Acceptance Criteria

1. [x] Implementar a tool `set_campaign_budget` no servidor MCP.
2. [x] A tool aceita: `customer_id`, `campaign_id` e `amount`.
3. [x] Conversão automática para `micros` (x 1.000.000) implementada.
4. [x] Busca automática do `resource_name` do orçamento vinculada à campanha.
5. [x] Sucesso validado em conta real (**TecPlaner**) com alteração para R$ 1,00.
