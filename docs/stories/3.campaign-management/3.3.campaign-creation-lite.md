# Story 3.3: Criação de Campanhas de Pesquisa (Draft)

**Epic:** 3 (Campaign Lifecycle)
**Status:** Done
**Role:** Developer
**Action:** Implementar a tool `create_search_campaign` para criação rápida de novas estruturas
**Benefit:** Permitir que agentes de IA escalem a conta criando novas campanhas baseadas em insights de discovery

---

## Acceptance Criteria

1. [x] Implementar a tool `create_search_campaign` no servidor MCP.
2. [x] A tool deve aceitar parâmetros básicos: `customer_id`, `campaign_name`, `daily_budget_amount`.
3. [x] Configuração padrão (Brasil/Português) para novas campanhas.
4. [x] Suporte a estratégias de lance automáticas (ex: `MAXIMIZE_CLICKS`).
5. [x] Retornar o ID da nova campanha e do orçamento criado.
